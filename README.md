# fila_bancaria
Este é um repositório destinado ao primeiro trabalho prático da disciplina: Programação para ciência de dados. O objetivo do trabalho é criar um simulador de fila bancária em python, com funcionalidades como inserção e remoção de clientes (seja via atendimento ou desistência), avanço de tempo, exibição da fila e cálculo de algumas estatísticas.
